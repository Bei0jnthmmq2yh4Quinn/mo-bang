# 写作任务标准流程

> 每次给老板写小红书文案时，所有模型（Claude/GPT/Gemini）统一遵守。

## 1. 必读文件
- `memory/writing-style.md` — 老板的写作风格档案（最高优先级）
- `memory/2026-02-22.md` — 素材/文案库（按需）

## 2. 写作流程
1. 小墨先读风格档案
2. 出初稿（或派子代理出稿时，把风格档案全文贴进 task 指令）
3. 发给老板
4. 老板口述/改写一版发回来
5. 小墨对比差异，更新 `memory/writing-style.md`
6. 下一篇按更新后的风格写

## 3. 子代理派活模板
```
你是小红书文案写手。请严格按照以下【风格档案】写作，这是客户的真实表达习惯：

【风格档案】
{粘贴 writing-style.md 全文}

【任务】
{具体写作要求}
```

## 4. 风格档案更新规则
- 每次老板给出改写版本后，必须更新
- 记录：用词变化、句式节奏、删减内容、新增内容、金句
- 累积越多轮，风格越准
