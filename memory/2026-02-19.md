## 经验教训
- SSH 长命令容易因引号问题 code 255，拆成小命令或 scp 脚本更稳
- mihomo SOCKS5 端口在 global 模式下可能不走代理节点，HTTP 端口才行
- 免费代理池节点大量共享出口 IP，需要 IP 去重逻辑
- GPTMail 有每日配额限制，备选 mail.tm 无需 key
