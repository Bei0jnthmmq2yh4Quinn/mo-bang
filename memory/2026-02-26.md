### [PROJECT:OpenClaw] Session model override 卡住问题
- **现象**: 用 `session_status(model=...)` 切模型后，系统自动切回 anyrouter/claude-opus-4-6
- **原因**: session 数据里有 `modelOverride` + `providerOverride` 持久字段，覆盖了默认模型
- **修复**: `session_status(model="default")` 清除 override
- **教训**: model switch 系统消息会设 override，需要用 `default` 显式清除
- **标签**: #openclaw #models #session
